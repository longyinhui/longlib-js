Create DataGrid

new DataGrid( id, div, design, source )

Description: Construct a new Data Grid by instantiating a DataGrid class.

id
Type: String
The unique identification of a Data Grid.

div
Type: String
The id attribute of a <div> tag, which will be the container of the Data Grid.

design
Type: String or JSON Object
The design of Data Grid, includes Data Grid's attributes, and its columns' definition, events, type, etc. (Please refer to var dataGridDesign in Demo part)

source
Type: String or JSON Object
The data source. (Please refer to var dataGridSource in Demo part)

Description: Add a new row of Data Grid.

position (optional)

Delete Row(s)

.deleteRow( rowIndex, [speed] )

Description: Delete a row of Data Grid.

rowIndex
Type: Int
The row index of the row to be deleted.

speed (optional)
Type: Int
The duration time of animation.

.deleteRows( rowIndexes, [speed] )

Description: Delete rows of Data Grid by indexes.

rowIndexes
Type: Array of Int
The row indexes of the rows to be deleted.

speed (optional)
Type: Int
The duration time of animation.
